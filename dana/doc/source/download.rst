Download
===============================================================================

There is currently no official dana package available and the subversion
repository houses the source trunk, documentation and examples.  You can browse
the repository from `here
<https://gforge.inria.fr/plugins/scmsvn/viewcvs.php/trunk/dana/?root=dana>`_ or
check out a working copy using an SVN client:

.. code-block:: none

   svn checkout svn://scm.gforge.inria.fr/svn/dana/trunk/dana dana-svn
